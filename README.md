# AAF to Ableton Converter

Convert AAF timeline exports from DaVinci Resolve into Ableton Live projects (.als).

**Live at:** https://aaf-to-ableton.vercel.app/

## How to use

1. Export an AAF from DaVinci Resolve (File → Export → AAF)
2. Drop the AAF file onto the converter
3. Download the ZIP
4. Unzip it — you'll get a folder containing the `.als` project file and a `Samples/imported/` folder
5. If your AAF had embedded audio, the WAV files are already inside `Samples/imported/`. If not, drag your audio files there manually
6. Open the `.als` file from inside the unzipped folder

## Privacy

Everything runs client-side in your browser. Your files are never uploaded to a server.
